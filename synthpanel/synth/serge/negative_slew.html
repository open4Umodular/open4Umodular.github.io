  • pcb pattern
  • photo/overlay

Note that there are a 1M and 120p capacitor wired in parallel in two locations, as marked 1M||120p.

PCB connections:
A = input (1)
B = output (1)
C = VC (1)
D = pulse out (1)
E = input (2)
F = output (2)
G = VC (2)
H = pulse out (2)
U = not used (0v)
W = 0v
X = +12v
Y = +6v
Z = -12v
