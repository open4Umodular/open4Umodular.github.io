PCB connections:
A = 3 + input
B = 3 ramp input
C = 3 output
D = 2 + input
E = 2 ramp input
F = 2 output
G = 1 + input
H = 1 ramp input
I = 1 output
J = Schmitt input
K = Schmitt output
U = not used (0v)
W = 0v
Y = +6v
