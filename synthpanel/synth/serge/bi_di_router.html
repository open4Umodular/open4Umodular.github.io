  • pcb pattern

PCB connections:
A = A1/A2 select input (1)
C = A2 (1)
B = A1 (1)
D = B (1)
E = A1/A2 select input (2)
F = A2 (2)
G = A1(2)
H = B (2)
I = A1/A2 select input (3)
J = A2 (3)
K = A1 (3)
L = B (3)
X = +12v
Z = -12v

There were extensive errors on the jack wiring marking of this module.
