PCB connections:
A = 1 output
B = 2 output
C = 3 output
D = 4 output
E = 5 output
F = 6 output
G = 7 output
H = 8 output
I = 9 output
J = 10 output
K = Clock input
L = Hold input
M = + to reset switch
N = From reset switch
O = Reset input
U = not used (0v)
W = 0v
Y = +6v
