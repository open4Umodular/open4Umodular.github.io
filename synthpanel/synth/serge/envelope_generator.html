PCB connections:
A = start input
B = Window pulse output
C = envelope output
D = End pulse output
E = cycle input
F = Hold input
G = Duration CV
H = Window size CV
Q = not used
U = not used (0v)
W = 0v
X = +12v
Y = +6v
