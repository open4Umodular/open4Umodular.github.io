Note that the later PCB looks to be identical to the 1973 version with the exception of the on-board pots. Neither match the schematic that can readily be found on the web.

PCB connections:
A = tws 1 input
B = tws 1 VC1
C = tws 1 VC2
D = tws 1 output
E = tws 2 input
F = tws 2 VC1
G = tws 2 VC2
H = tws 2 output
I = tws 3 input
J = tws 3 VC2
K = tws 3 VC1
L = tws 3 output

U = 0v
x = +12v
W = 0v
